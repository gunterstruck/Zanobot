# PWA Icons

Dieser Ordner enthält die Icons für die Progressive Web App.

## Benötigte Icons

Die folgenden Icon-Größen werden vom `manifest.json` referenziert:

- `icon-72.png` - 72x72px
- `icon-96.png` - 96x96px
- `icon-128.png` - 128x128px
- `icon-144.png` - 144x144px
- `icon-152.png` - 152x152px
- `icon-192.png` - 192x192px (maskable)
- `icon-384.png` - 384x384px
- `icon-512.png` - 512x512px (maskable)

## Icon-Design

Das Icon sollte:
- Das Zanobot-Logo (Kopf mit Gehirn + Schallwellen) zeigen
- Auf dunklem UND hellem Hintergrund funktionieren
- Die Primärfarbe #00D4FF verwenden
- Im Maskable-Format einen Safe-Area-Padding haben (20% Abstand)

## Generierung

Icons können mit Tools wie:
- [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- Photoshop/Figma + Export

generiert werden.
