# Zanobot PWA - Static Demo

This is a minimal, static version of the Zanobot PWA for design testing.

## Features
- ✅ Pure HTML/CSS/JS (no build tools)
- ✅ Responsive design
- ✅ Theme switching (Neon, Light, Brand)
- ✅ PWA-ready with service worker
- ✅ Works immediately on GitHub Pages

## Demo Features
- Collapsible sections
- Theme toggle
- Mockup data
- All UI elements visible

## Deploy to GitHub Pages
1. Create new repository
2. Push all files
3. Enable GitHub Pages on main branch
4. Done! No build step needed.
